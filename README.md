# MEAN To-Do App

A full-stack task management application built with the MEAN (MongoDB, Express.js, Angular, Node.js) stack.  
Users can create, update, and delete tasks with real-time UI updates and a clean, responsive interface.

## Features
- Add, Edit, Delete To-Do Items
- RESTful API built with Node.js + Express.js
- Persistent data storage using MongoDB
- Frontend built using Angular
- Modular and scalable folder structure

## Tech Stack
**Frontend:** Angular  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  
**Version Control:** Git & GitHub

## Running the project
cd backend
npm install
npm start

cd frontend
npm install
ng serve
